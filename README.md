## Tech Stack

Frontend    - React, JavaScript
Backend     - Django DRF, python
Database    - PostgreSQL, Supabase
Server      - Netlify for frontend, Render for backend

## Installation

Clone project:

```shell
git clone https://github.com/aliendevuz/uzblog.git
cd uzblog
```

Frontend setup:

```shell
cd frontend
npm install
npm run dev
```

Backend setup:

```shell
cd backend
pip install -r requirements.txt
python manage.py runserver
```
